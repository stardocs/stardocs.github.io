Stardog Docs

Introduction

Stardog is a fast, lightweight, commercial RDF database for mission-critical apps: it supports SPARQL 1.1; HTTP and the SNARL protocol for remote access and control; RDF as a data model; OWL 2 and rules for inference and data analytics.

Acquiring Stardog & Support

Download Stardog to get started. The Stardog support forum, stardog@clarkparsia.com, is the place to report bugs, ask questions, etc. You can also ask questions on Stack Overflow using the tag stardog.

Contributing

There are several open source components of Stardog; feel free to submit pull requests: stardog-docs, stardog.js, stardog-groovy, stardog-spring, and stardog.rb.
